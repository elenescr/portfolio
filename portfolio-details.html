Personal Finance Dashboard – TBC & BOG Transactions (Power BI)

Designed and developed an interactive Power BI dashboard to analyze one year of personal banking transactions from TBC and BOG accounts. The project involved cleaning and transforming data from different formats, unifying currency values, and categorizing expenses based on description patterns. Key insights include monthly spending trends, income vs. expense breakdown, category-wise analysis, average daily spending, and bank-wise comparisons. The dashboard features slicers, dynamic visuals, and tooltips for user-friendly interaction. This project demonstrates skills in data modeling, Power Query, DAX, and effective dashboard design for real-world financial data.
